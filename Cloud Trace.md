### **Core Function**

Tracing system that **collects latency data** from distributed applications and displays it in Google Cloud console.

### **Platform Support**

- **App Engine** applications
- **Compute Engine** VMs
- **Google Kubernetes Engine** containers
- **Android devices**

### **Key Features**

#### **Latency Analysis**

- Analyze changes to applications' **latency profiles**
- View performance insights in **near-real time**
- Track performance changes through Google Cloud Console

#### **Automated Reporting**

- **Automatically analyze** all application traces
- Generate **in-depth latency reports**
- **Surface performance degradations**
- **Continuously gather and analyze** trace data
- **Automatically identify recent changes** to application performance

### **Primary Use Case**

Identify and analyze latency issues in distributed applications before they impact users.
